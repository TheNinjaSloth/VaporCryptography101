Rail Fence Definition

Rail Fence (also called a zigzag cipher) is a transposition cipher. It derives its name from the way in which it is encoded. In the rail fence cipher, the plaintext is written downwards and diagonally on successive "rails" of an imaginary fence, then moving up when we reach the bottom rail.

When creating the railfence cypher, we need a message. Say the message as x number of letters. Typically, we then create a 3 by x sized grid. Within this grid, in the first column, you then pick a row to start in. If you pick the top, then you go down diagonally until you reach the bottom, at which point you you go back up diagonally, and repeat until your message is complete. The same applies for the third row, but vice versa. In the event you pick the second, you will pick which direction you want the message to proceed in. Once this grid is made, you read encrypt the message by going left to right, top to bottom, row by row. This will scramble your message. Write it out as one garbled word. When decrypting, reverse the process.
